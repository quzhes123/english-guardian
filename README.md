# 英语守护者 - 微信小程序

> 专为初中生打造的中考英语词汇背诵小程序

## 功能特点

- 📚 **中考词汇表** - 收录天津近5年中考高频词汇，按字母顺序排列
- 📅 **每日任务** - 每天背诵20个新词 + 复习前一天的词
- ✅ **打卡监督** - 每日打卡记录，培养学习习惯
- 🔁 **智能复习** - 遵循艾宾浩斯遗忘曲线提醒复习
- 📝 **词汇测试** - 看中文写英文 / 看英文选中文
- 📊 **进度统计** - 可视化学习进度和坚持天数

## 目录结构

```
english-guardian/
├── app.js              # 小程序主逻辑
├── app.json            # 小程序配置
├── app.wxss            # 全局样式
├── data/
│   └── vocabulary.js   # 中考词汇数据
├── images/             # 图标资源
├── pages/
│   ├── index/           # 首页
│   ├── study/           # 学习页
│   ├── review/         # 复习页
│   ├── word-list/      # 词汇表
│   ├── test/           # 测试页
│   ├── stats/          # 统计页
│   └── mine/           # 我的页面
└── project.config.json # 项目配置
```

## 使用说明

1. 打开微信开发者工具
2. 导入此项目
3. 修改 app.json 中的 appid 为你的 appid
4. 编译运行

## 自动部署

本项目支持 GitHub Actions 自动部署。配置 secrets 后，推送代码即可自动构建并上传到微信。

```yaml
# 需要配置的 Secrets
WX_APPID      # 微信小程序 AppID
WX_PRIVATE_KEY # 微信小程序私钥
```
